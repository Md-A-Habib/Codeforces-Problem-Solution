# 🚀 Codeforces Problem Solutions

This repository contains my personal solutions to various **Codeforces problems**, written in **C++**.  
It is part of my journey to improve problem-solving skills and excel in **Competitive Programming (CP)**.

## 📌 About

Codeforces is one of the most popular competitive programming platforms where programmers solve algorithmic problems and participate in contests.

In this repository, I solve problems ranging from **800–1800+ rated difficulty**, focusing on:

- Data Structures
- Algorithms
- Greedy Techniques
- Mathematics
- Number Theory
- Binary Search
- Graph Algorithms
- Dynamic Programming (DP)
